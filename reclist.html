Resource List

*IN PROGRESS* I learn best through interacting with the material, so I've been searching for interactive resources associated with my studies. Here are some resources I have found useful for my Mechanical Engineering courses so far.

Seeing Theory

This open access website has interactive tools to help visualize basic concepts in probability and statistics.

Timothy Philpot's MecMovies

Interactive demonstrations, games and practice exercises for broad material mechanical properties. Also see MDSolids.

Thermo Jesus

Mobile app for iOS, contains practice problems with conceptual breakdowns and solutions, water calculator and practice quizzes.
