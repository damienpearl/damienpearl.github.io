Project Overview

Project: using emPRISE lab's compliant control package to allow robot arm to be manipulated by a user, system records user-created motions into rosbags and stores them in a library. this library of rosbags can then be accessed and replayed.

Features

  • Record motion in real-time
  • Replay recorded motion
  • Save and load motion data
