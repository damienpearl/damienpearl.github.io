Undergraduate Research Assistant

Explored how humans interpret robot motion by designing a study pipeline from data collection to interpretation modeling.

  • Built a ROS1 + Python system to generate, record, and replay precise robotic movements for consistent data capture.
  • Ran a 30-person study and surfaced significant trends in how participants associate motion with perceived robot intent.
  • Used Whisper AI and open coding to label participant reasoning from study audio, refining early-stage behavioral data.
  • Analyzed links between user-designed robot movements and expert ratings using BERT and KMeans clustering.
